Project: KarplusSynth
Language: C++17
Build system: CMake >= 3.14

Dependencies:
1. PortAudio
   - GitHub: https://github.com/PortAudio/portaudio
   - Used for real-time audio playback

2. libsndfile
   - GitHub: https://github.com/libsndfile/libsndfile
   - Used for reading/writing audio files

Build requirements:
- C++17 compatible compiler (e.g. GCC >= 7.5, Clang >= 6.0, MSVC 2019+)
- CMake 3.14 or newer

Notes:
- On Linux both libraries are automatically fetched and built via CMake (FetchContent).
- On Windows, PortAudio and libsndfile require DLL files to be present in the same folder as the executable.
